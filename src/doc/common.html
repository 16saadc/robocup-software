RoboJackets - RoboCup

common

To use anything in common just include the provided CMakeLists.txt in your CMakeLists.txt file. If you wish to use one of the particular libraries set the appropriate variable to 'true' in your CMakeLists.txt file.

Files

Sizes.h
Various object sizes (field, ball, robot, etc...) all in stnadard units
Team.h
The Team enum with Yellow and Blue items. Used to distinguish team which team we are receiving/sending packets from/to.

Libraries

Geometry

set (USE_GEOMETRY true) to activate
${GEOMETRY} variable in the target_link_libraries to use

Library with basic geometry classes. Look at header files for interface documentation.

Packet

set (USE_PACKET true) to activate
${PACKET} variable in the target_link_libraries to use

Library for interprocess communication.

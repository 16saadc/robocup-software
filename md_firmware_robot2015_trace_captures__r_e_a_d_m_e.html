Logic Analyzer Trace Captures

This directory contains logic analyzer data exports for different areas of the RoboCup robot. The Saleae *Logic16 USB Logic Analyzer* was the device used to capture the traces. The computer used for interfacing with the Logic16 was running Ubuntu 14.04. The software used was Saleae's *Logic* program (1.1.34). The files with a *.logicdata extension are the saved files from the *Logic* program. The corresponding *.csv files contain the raw data from the captured waveform.

The program's configuration file is included, but it is not needed for properly viewing the *.logicdata files

It is highly recommended to use the *Logic* program for viewing the waveforms from the original *.logicdata files.

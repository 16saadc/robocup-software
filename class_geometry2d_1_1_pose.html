Detailed Description

Position with orientation Uses:

  • robot pose (not velocity)
  • checks for ball given a direction (useful for moving balls) Utility functions allow for detecting quadrants where a ball might be

Note: this is not a velocity, as the angle is fixed to be on [-pi, pi]

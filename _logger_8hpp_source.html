00001 // This logger implements a circular buffer for recent history and writes all
00002 // frames to disk.
00003 //
00004 // _history is a circular buffer.
00005 //
00006 // Consider a sequence number for each frame, where the first frame passed
00007 // to addFrame() has a sequence number of zero and the sequence number is one greater for
00008 // each subsequent frame.
00009 //
00010 // _nextFrameNumber is the sequence number of the next frame to be stored by addFrame().
00011 //
00012 // lastFrame() returns the sequence number of the latest available frame.
00013 // It returns -1 if no frames have been stored.
00014 //
00015 // You can get a copy of a frame by passing its sequence number to getFrame().
00016 // If the frame is too old to be in the circular buffer (or the sequence number is
00017 // beyond the most recent available) then getFrame() will return false.
00018 //
00019 // Frames are allocated as they are first needed.  The size of the circular buffer
00020 // limits total memory usage.

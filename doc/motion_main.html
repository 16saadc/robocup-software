RoboJackets - RoboCup

main.cpp

The standard main file for handling cmd line arguments and starting the program. Below is its usage.
Usage: [-b] [-y] [-a] <config file>
-b: enable blue team control.
-y: enable yellow team control.
-a: default to auto mode when a controller is plugged in.
You can only run motion for either the blue or the yellow team.

void usage(const char* prog)
Usage function. Is called for either incorrect cmd line input or with the -h flag
int main(int argc, char *argv[])
Creates a new QCoreApplication, processes cmd line input, creates a new ConfigFile object and loads. Also, creates a MotionController for either the yellow or blue team and starts it.

Detailed Description

The CC1201 class handles wirelessly sending and receiving data using the TI CC1201 radio transceiver.

The radio is configured via a collection of registers. These control things such as transceiver frequency, packet error checking, packet length, among MANY other things.

The best resource for learning more about the radio and it's SPI interface is the User Guide on TI's website: http://www.ti.com/lit/ug/swru346b/swru346b.pdf

Initialize the CC1201 with the given SPI, chip select (inverted), and interrupt line pins.

The radio is configured by providing an array of registerSetting_t's which specify (address, value pairs) for configuration registers. Typically the register settings files are exported from TI's SmartRF Studio program.
